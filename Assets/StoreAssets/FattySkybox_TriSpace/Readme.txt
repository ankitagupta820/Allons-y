Fatty Sky - TriSpace. (2020/02/23 - by Fatty War, id3644@gmail.com)


This document explains how to use the pack.

Introduction
	
	A polygon universe suitable for the lowpoly style.
	This package contains stylized four space skyboxes.
	
		1. Emtpy space(+asteroid).
		2. Gas cloud space.
		3. Near earth(+moon).
		4. All stars-Solar System 8(9) planet.
		5. All stars (+Gas Cloud).

	* Planet is embedded in a texture.
	* Each sky is panorama type and box type.
	* Includes a lens flare effect instead of the sun (1 type).



	1-1. File Naming : 
	
	* FattySky50 + Emtpy = FattySky50_Emtpy
	* (a + b)
	
	a : Package name.
	
	b : skybox name.
		1. Emtpy.
		2. GasCloud.
		3. NearEarth.
		4. AllStars.
		5. AllStarsCloud.
	
	1-2. Materials
		*SkyBox (Assets\FattySkybox_TriSpace\Materials\SkyBox)
			-Box type sky material is here.

		*SkyPanoramic (Assets\FattySkybox_TriSpace\Materials\SkyPanoramic)
			-Panorama type sky material is here.

	1-3. Textures
		*SkyBox (Assets\FattySkybox_TriSpace\Textures\SkyBox)
			-Box type sky texture is here.
			-Sky box texture is 6 bundles.(front, back, left, right, top, down)

		*SkyPanoramic (Assets\FattySkybox_TriSpace\Textures\SkyPanoramic)
			-Panorama type sky texture is here.
